# 마크다운 데이터 변환 가이드

## 현재 상황 분석

### 실제 파일명 형식
- **MP3**: `Balad_1.mp3`, `Balad_2.mp3` (숫자 기반)
- **이미지**: `A glass of soju.png` (공백 포함, 실제 노래 제목)

### 마크다운 데이터 형식
- **MP3**: `A-glass-of-soju.mp3` (하이픈)
- **이미지**: `A-glass-of-soju.png` (하이픈)

## 문제점

1. **MP3 파일명 불일치**
   - 마크다운: `A-glass-of-soju.mp3`
   - 실제: `Balad_1.mp3`
   - **해결**: `title`을 기준으로 실제 파일명 찾기 필요

2. **이미지 파일명 불일치**
   - 마크다운: `A-glass-of-soju.png` (하이픈)
   - 실제: `A glass of soju.png` (공백)
   - **해결**: 하이픈을 공백으로 변환하거나 유연한 매칭 필요

## 해결 방안

### 옵션 1: 파일 시스템 스캔 + title 매칭 (권장)
1. 실제 파일 시스템을 스캔
2. `title`을 기준으로 실제 파일명 찾기
3. 하이픈/공백/언더스코어를 무시하고 유연하게 매칭

### 옵션 2: 파일명 변경
- MP3: `Balad_1.mp3` → `A-glass-of-soju.mp3`
- 이미지: `A glass of soju.png` → `A-glass-of-soju.png`
